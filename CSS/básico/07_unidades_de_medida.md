# Unidades de medida

## Unidades de medida absolutas

- **cm**: centímetro
- **mm**: milímetro
- **in**: polegadas
- **_px_**: pixels (recomendado)
- **pt**: pontos
- **pc**: picas

## Unidades de medida relativas

- **_%_**: porcentagem (recomendado)
- **vw**: viewport width
- **vh**: viewport height
- **_em_**: em relação ao tamanho padrão do pai (recomendado)
- **_rem_**: em relação ao tamanho padrão do root (recomendado)
